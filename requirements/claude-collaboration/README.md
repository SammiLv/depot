# Claude 多终端协作文档索引

这个目录用于沉淀当前项目的 Claude 多终端并行开发方案、提示词模板和执行规范。

当前推荐模式：**默认 3 终端，按本轮菜单动态分工。**

- 终端 A：菜单 1
- 终端 B：菜单 2
- 终端 C：平台 / 公共 / DB 协调
- 仅在必要时增加终端 D：DB

## 文档清单

### 1. 启动清单
- [claude-多终端并行开发启动清单.md](./claude-多终端并行开发启动清单.md)
- 用途：快速了解默认 3 终端模型、菜单式分工、独立 worktree + 分支协作方式和开发规则。

### 2. 精简提示词模板
- [claude-多终端精简提示词模板.md](./claude-多终端精简提示词模板.md)
- 用途：给菜单终端、平台终端、DB 终端直接复制使用。

### 3. 分支命名规范与合并顺序说明
- [claude-分支命名规范与合并顺序说明.md](./claude-分支命名规范与合并顺序说明.md)
- 用途：统一菜单式分支命名、DB 改动时的协作方式、以及多分支合并顺序。

### 4. 首次落地执行手册
- [claude-首次落地执行手册.md](./claude-首次落地执行手册.md)
- 用途：第一次正式启用“两个菜单终端 + 一个平台终端”时的 SOP。

### 5. 本地双环境运行指南
- [claude-本地双环境运行指南.md](./claude-本地双环境运行指南.md)
- 用途：快速确认公司电脑与家里电脑分别执行哪个命令。

### 6. GitHub 同步极简清单
- [claude-GitHub同步极简清单.md](./claude-GitHub同步极简清单.md)
- 用途：快速确认哪些文件必须同步到 GitHub，换电脑后怎么继续。

### 7. 跨电脑协作记忆
- [claude-跨电脑协作记忆.md](./claude-跨电脑协作记忆.md)
- 用途：沉淀需要跨电脑长期保留的稳定协作规则，不依赖 Claude 本机记忆。

### 8. 合并后环境刷新脚本清单
- [claude-合并后环境刷新脚本清单.md](./claude-合并后环境刷新脚本清单.md)
- 用途：主仓库或各分支合并主仓库最新代码后，快速执行依赖安装、Prisma 同步、build 和服务重启。
